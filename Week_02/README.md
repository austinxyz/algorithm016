### 树，二叉树，二叉搜索树
- 二维结构，为什么有树，来源于现实
- 二叉树，如果只有一个儿子，退化为链表
- 二叉搜索树， 左子树所有值都比根大，右子树所有值都比根小，插入，删除 O（logN）
- 树通常用递归来解，没法用循环来遍历
- 前，中，后，是说根节点的位置
